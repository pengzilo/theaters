<p><strong>2.无法获取到资源：可能是因为目标网站对IP进行了限制或者资源接口地址已经更换新的接口地址，资源网接口地址失效等等，需要重新到资源帮助中心获取该接口地址；</strong></p>

<p><strong>4.采集速度慢：可能是因为网络环境不佳，需要优化网络环境。</strong></p>
